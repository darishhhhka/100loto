# Electron

##### Положение и размер окна
- `--width={число}` - ширина окна
- `--height={число}` - высота окна
- `--offsetX={число}` - положение окна на экране
- `--offsetY={число}` - положение окна на экране
- `--display={число}` - индекс монитора, на котором выводить приложение
- `--fullscreen` - открывать окно на весь экран
- `--resizable` - можно ли изменять размер окна

##### Запуск приложения
###### Сервер
- `--localhost` - открывать локальный сервер с дебаг версией
- `--server` - использовать локальный сервер 
- `--port={число}` - порт на котором запущен дебаг сервер (по умолчанию 3000) или на котором запускать 
- `--devTools` - открыть devTools при запуске
- `--watch` - перезагрузка страницы при обновлении файлов 

###### Приложение
- `--app={строка}` - название файла (без расширения) который открывается при запуске 
- `--data={строка}` - GET параметр data. Используется для передачи пути к файлу с данными приложения
- `--debug={строка,строка}` - GET параметр debug. Используется для включению определенных дебаг функций
- `--queryParams={ключ=значение&ключ2=значение2}` - дополнительные GET параметры
